# 📊 Portafolio de Análisis de Datos

Este repositorio contiene mis proyectos principales de Análisis de Datos, documentados con metodologías de descubrimiento de conocimiento (KDD), dashboards interactivos y presentaciones ejecutivas.

---

## ☕ Proyecto #1: Análisis de Datos - Coffee Sales
Este proyecto analiza el comportamiento del consumidor, simulaciones de paquetes de productos (bundling) y proyecciones de ingresos para optimizar las ventas de una cadena de café.

* 📈 **[Diapositivas Resumen](https://docs.google.com/presentation/d/19_nWRm0_zapuBiCeAzJsQNZde0cLUvTenAIGy-R30H0/present?slide=id.g3a87eabab9d_2_75)**: Presentación ejecutiva del proyecto.
* 📊 **[Dashboard Interactivo](https://docs.google.com/spreadsheets/d/1iPTQq2u4AXH7E_DAI1QHkyMdYQFEBwzKbyc2mfYJry8/template/preview)**: Reporte limpio optimizado para presentación (Hoja Pivot Table 12).
* 🛠️ **[Metodología Aplicada (KDD)](https://docs.google.com/presentation/d/19_nWRm0_zapuBiCeAzJsQNZde0cLUvTenAIGy-R30H0/present?slide=id.g3a87eabab9d_4_16)**: Proceso de selección, limpieza y transformación de datos.
* 💾 **[Data Set Raíz](https://docs.google.com/spreadsheets/d/1iPTQq2u4AXH7E_DAI1QHkyMdYQFEBwzKbyc2mfYJry8/preview?gid=528103202&single=true)**: Acceso directo a la base de datos utilizada para el análisis.
